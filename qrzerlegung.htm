Anzahl Nachkommastellen:

Die Householder Transformation ist eine Spiegelung, so dass gewünschte Stellen zu Null werden. Die Givens Rotation ist als Drehung ein Spezialfall der Householder Transformation. Das Ergebnis zeigt Q*A = R. R ist eine rechte obere Dreiecksmatrix, Q ist eine orthogonale Matrix. Dies kann umgestellt werden zu A = Q(transponiert)*R. Das Verfahren ist sehr stabil.
